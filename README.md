# SmartFit - 智能穿搭助手

一款基于 AI 的智能穿搭推荐系统，采用极简主义设计风格，为用户提供个性化的穿搭建议。

## ✨ 核心功能

### 1. 用户资料设置
- 基础信息：性别、身高、体重、三围
- 外貌特征：肤色、体型
- 照片上传（用于虚拟试衣）

### 2. 色彩季型测试 ⭐
- 5 道专业色彩测试题
- 智能诊断用户的色彩季型（春/夏/秋/冬）
- 推荐适合的颜色和避免的颜色
- 提供穿搭配色建议

### 3. 我的衣橱 ⭐
- 添加和管理现有衣物
- 按类型分类（上装/下装/外套/鞋/配饰）
- 记录颜色和风格标签
- 支持照片上传

### 4. 每日穿搭推荐
- 基于天气自动推荐
- 场合选择（通勤/约会/运动/街头）
- 结合色彩季型和衣橱内容
- AI 生成穿搭效果图（Mock 版）

### 5. 虚拟试衣 ⭐
- 上传个人照片
- 选择想试穿的衣物
- 预览穿搭效果（演示版）

### 6. 商品推荐
- 基于用户季型推荐商品
- 按类型筛选
- 一键跳转淘宝搜索

## 🎨 设计特色

### 极简黑白灰配色
- 主色：纯黑 `#000000`
- 背景：浅灰 `#f8f8f8`
- 边框：灰色 `#e5e5e5`
- **无渐变色、无花哨装饰**

### 设计语言
- ✅ 直角边框（无圆角）
- ✅ 1px 描边
- ✅ 等宽字体（Mono）
- ✅ 充足留白
- ✅ 网格布局
- ❌ 禁止渐变
- ❌ 禁止鲜艳背景

## 🛠️ 技术栈

- **框架**: Next.js 16 (App Router)
- **UI**: React 19 + TailwindCSS 4
- **动画**: Framer Motion
- **图标**: Lucide React
- **数据**: LocalStorage (本地持久化)

## 📱 使用流程

1. **首页** - 了解功能，点击"开始使用"
2. **用户资料** - 填写身材数据和上传照片
3. **色彩测试** - 完成 5 道测试题
4. **测试结果** - 查看你的色彩季型和建议
5. **我的衣橱** - 添加现有衣物
6. **每日推荐** - 选择场合，生成穿搭方案
7. **查看结果** - 浏览穿搭效果图和商品推荐
8. **虚拟试衣** - 试穿想买的衣物（演示版）

## 🔑 核心逻辑

### 色彩季型测试算法
基于 5 道问题，计算春夏秋冬四个季型的得分，选择最高分为用户季型。

### 三围估算公式
- 胸围 = 身高 × 0.54
- 腰围 = 身高 × 0.38
- 臀围 = 身高 × 0.54 + 2

### Mock 数据说明
当前版本使用 Mock 数据：
- 天气：固定 22°C 晴天
- 穿搭图片：Unsplash 图库
- 衣橱：预置 8 件衣物
- 商品：预置 10 件推荐商品
